wiCollapse
directive in module dist.ui.collapse

Description

wiCollapse 用于实现元素的折叠展开操作，若元素需指定高度，请在 style 中定义 height，否则无法获取，将默认为 auto。
IE9 及以下版本不支持动画；此外，若将 height 设置为 'calc(……)'，切换时将跳过动画。

Usage

as attribute
<ANY wi-collapse="{boolean}">
   ...
</ANY>

Parameters

ParamTypeDetails
wiCollapseboolean

true 时折叠，false 时展开。
