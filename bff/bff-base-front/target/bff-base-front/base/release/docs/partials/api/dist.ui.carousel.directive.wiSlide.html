wiSlide
directive in module dist.ui.carousel

Description

创建一个slide，只能作为 wiCarousel 的子指令使用，另见：wiCarousel.

Usage

as element:
<wi-slide
       [active="{boolean}"]>
</wi-slide>
as attribute
<ANY wi-slide
     [active="{boolean}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
active
(optional)
boolean

模型绑定, 设定 slide 当前是否为激活状态.
