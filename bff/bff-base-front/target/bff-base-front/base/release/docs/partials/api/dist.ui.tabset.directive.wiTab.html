wiTab
directive in module dist.ui.tabset

Description

单个标签页定义标签，只能在wiTabset标签内使用

Usage

as element:
<wi-tab
       [wiid="{string}"]
       [heading="{string}"]
       [active="{boolean}"]
       [disable="{boolean}"]
       [regular="{boolean}"]
       [icon="{string}"]
       [src="{string}"]
       [size="{number|length}"]>
</wi-tab>

Parameters

ParamTypeDetails
wiid
(optional)
string

标签标识id

heading
(optional)
string

标签头内容，可以是一个字符串或一段HTML

active
(optional)
boolean

标签初始时是否被选中，默认为 false，若要通过其他方式改变 active 的值，需为 active 指定有意义的初始值

disable
(optional)
boolean

标识标签是否禁用，默认为 false。
wi-tabset 若未定义 closeable=false，此属性失效。

regular
(optional)
boolean

是否是固定标签，默认为false。添加此属性且不设置为 false，则为固定 tab，不可关闭。
wi-tabset 若定义 closeable=false，此属性失效。

icon
(optional)
string

标签页图标的 url

src
(optional)
string

标签页内容链接地址，若定义此属性，wi-tabset 中原有的

size
(optional)
numberlength

组件宽度（纵向 tabset 为高度），默认由内容撑开。
number 将自动添加单位 px。
length 为 number 接长度单位（相对单位和绝对单位）。
相对单位：em, ex, ch, rem, vw, vh, vm, %
绝对单位：cm, mm, in, pt, pc, px
