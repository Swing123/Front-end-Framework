wiAccordion
directive in module dist.ui.accordion

Description

wiAccordion是一个折叠面板，如果给定显示区域比较小，就可以考虑使用这个组件。

Usage

as element:
<wi-accordion
       [wiid="{object}"]
       [width="{number|length}"]
       [height="{number|length}"]
       [headheight="{number}"]
       [oneopen="{boolean}"]
       [onselect="{function}"]
       [onunselect="{function}"]>
</wi-accordion>
as attribute
<ANY wi-accordion
     [wiid="{object}"]
     [width="{number|length}"]
     [height="{number|length}"]
     [headheight="{number}"]
     [oneopen="{boolean}"]
     [onselect="{function}"]
     [onunselect="{function}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
wiid
(optional)
object

若定义了此属性，可供调用接口。
.element() 获取当前 accordion 对应的 jqlite 元素，
.options() 获取当前用户设置：{ width,height }，
.panels() 获取所有面板对应的 scope 集合。
.getSelect() 获取选中的面板对应的 scope。
.getGroup(which) 获取指定 heading或index 的面板对应的 scope。
.toggle(which) 切换指定 heading或index 的面板的选中状态。
.reCompute() 重新适应尺寸（当其祖先元素由隐藏改为显示时，可能需要手动调用此方法，以重新计算尺寸）。

width
(optional)
numberlength

宽度，用来设定整个accordion的宽度，不指定宽度则面板的宽度是父容器的100%。
number 将自动添加单位 px。
length 为 number 接长度单位（相对单位和绝对单位）。
相对单位：em, ex, ch, rem, vw, vh, vm, %
绝对单位：cm, mm, in, pt, pc, px

height
(optional)
numberlength

高度, 指定此属性后，每个折叠面板都拥有同样的高度，不指定则每个面板的高度是根据内容自适应的。
说明同 width。

headheight
(optional)
number

标题栏高度的数值。不带单位“px”，默认为 32。

oneopen
(optional)
boolean

是否只能打开一个 accordion，默认为 false，若定义了 height，此属性默认为 true，且用户设置不生效。

onselect
(optional)
function

打开面板后的回调方法，参数：index(当前面板索引位置),scope(面板对应的 scope)。

onunselect
(optional)
function

关闭面板后的回调方法，参数：index(当前面板索引位置),scope(面板对应的 scope)。
