wiCombobox
directive in module dist.ui.combobox

Description

下拉框

Usage

as element:
<wi-combobox
       [dataprovider="{array}"]
       [selecteditem="{object}"]
       [dropdownwidth="{pixels}"]
       [labelfield="{string}"]
       [groupfield="{string}"]
       [rowcount="{string}"]
       [editable="{boolean}"]
       [multiselect="{boolean}"]
       [enable="{boolean}"]
       [width="{number|length}"]
       [itemchange="{function}"]>
</wi-combobox>

Parameters

ParamTypeDetails
dataprovider
(optional)
array

数据源.

selecteditem
(optional)
object

默认选中条目.

dropdownwidth
(optional)
pixels

弹出框宽度(不含单位：px)。未定义时，若定义了纯数字的 width,默认为 width,否则默认由文本自动撑.

labelfield
(optional)
string

显示字段，默认为"name".

groupfield
(optional)
string

分组字段.

rowcount
(optional)
string

显示行数，默认为"5".

editable
(optional)
boolean

是否可输入，默认为"true".

multiselect
(optional)
boolean

多选.

enable
(optional)
boolean

是否可用.

width
(optional)
numberlength

组件宽度，默认为 150px。
number 将自动添加单位 px。
length 为 number 接长度单位（相对单位和绝对单位）。
相对单位：em, ex, ch, rem, vw, vh, vm, %
绝对单位：cm, mm, in, pt, pc, px

itemchange
(optional)
function

选中条目改变事件.
