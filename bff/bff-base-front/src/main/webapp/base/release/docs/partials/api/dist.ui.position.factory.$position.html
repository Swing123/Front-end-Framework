$position
service in module dist.ui.position

Description

$position 提供一套获取元素位置，计算元素位置的方法。

Methods

  • getZIndex()

    获取最大 z-index，使指定元素在顶层展现，从 5000 开始。

    Returns
    number

    z-index 数值。

  • offset(element)

    获取 element 相对于文档的偏移，参考 jQuery 中的 element.offset()。

    Parameters
    ParamTypeDetails
    elementelement

    要计算的元素 - jqLite 元素。

    Returns
    object

    {width:.., height:.., top:.., left:.., bottom:.., right:..} - 属性值是单位为 px 的数字。

  • offsetTop(element)

    获取 element 相对于顶层 window 的偏移。

    Parameters
    ParamTypeDetails
    elementelement

    要计算的元素 - jqLite 元素。

    Returns
    object

    [viewElPos(相对顶层 window 的偏移), domElPos(相对顶层 window 的 body 的偏移)]
    viewElPos、domElPos：{width:.., height:.., top:.., left:.., bottom:.., right:..} - 属性值是单位为 px 的数字。

  • position(element)

    获取 element 相对于父元素（定位元素）的偏移，参考 jQuery 中的 element.position()。

    Parameters
    ParamTypeDetails
    elementelement

    要计算的元素 - jqLite 元素。

    Returns
    object

    position {width:xx, height:xx, top:xx, left:xx, bottom:xx, right:xx} - 属性值是单位为 px 的数字。

  • positionElements(refEl, targetElWidth, targetElHeight, positionStr)

    根据 referEl 计算 targetEl 相对于父定位元素的位置。

    Parameters
    ParamTypeDetails
    refElelement

    触发元素 - jqLite 元素。

    targetElWidthnumber

    要定位的元素的宽(单位：像素)。

    targetElHeightnumber

    要定位的元素的高(单位：像素)。

    positionStrstring

    定位方向: p-p，可能的值：top,left,right,bottom,center，但一级弹出方向不支持 center。

    Returns
    position

    {top:.., left:..} - 属性值是单位为 px 的数字。

  • positionElements(referEl, targetElWidth, targetElHeight, positionStr, adaptable, appendToBody)

    根据 referEl 的位置，弹出项的尺寸，弹出方向，计算 弹出项的位置。

    Parameters
    ParamTypeDetails
    referElelement

    参照元素，根据其位置确定弹出项位置 - jqLite 元素。

    targetElWidthnumber

    要定位的元素的宽(单位：像素)。

    targetElHeightnumber

    要定位的元素的高(单位：像素)。

    positionStrstring

    定位方向: p-p，可能的值：top,left,right,bottom,center。

    adaptableboolean

    是否自适应，根据浏览器可视部分空间调整弹出方向。

    appendToBodyboolean

    弹出项是否是 的直接子元素，若为 true,位置相对于文档，若为 false，位置相对于定位父元素。

    Returns
    array

    [
    {top:'..px', bottom:'..px', left: '..px', right:'..px'}, - 属性值是单位为 px 的数字
    positionStr - 'top-left'/'top-right'/'bottom-left'/'bottom-right'/'top-left'/'top-right'/'bottom-left'/'bottom-right'
    ] - 属性值是单位为 px 的数字。
