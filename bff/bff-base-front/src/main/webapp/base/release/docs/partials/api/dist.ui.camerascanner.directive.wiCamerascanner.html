wiCamerascanner
directive in module dist.ui.camerascanner

Description

拍照上传

Usage

as element:
<wi-camerascanner
       [url="{string}"]
       [to-pdf="{boolean}"]
       [upload-complete="{function}"]>
</wi-camerascanner>

Parameters

ParamTypeDetails
url
(optional)
string

上传地址.

toPdf
(optional)
boolean

是否转pdf，默认为true.

uploadComplete
(optional)
function

上传完成回调.
