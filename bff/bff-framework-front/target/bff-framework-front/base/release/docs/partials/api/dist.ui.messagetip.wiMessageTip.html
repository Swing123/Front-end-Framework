wiMessageTip
service in module dist.ui.messagetip

Description

消息提示窗口

主要功能：
（1）支持自定义弹出位置
（2）支持内容自定义HTML

Methods

  • closeOne(id)

    关闭消息提示窗口

    Parameters
    ParamTypeDetails
    idString

    窗口id

  • open(options)

    打开消息提示窗口

    Parameters
    ParamTypeDetails
    optionsObject

    {
    width : Number类型，[可选]窗口宽度，默认自适应
    height : Number类型，[可选]窗口高度，默认自适应
    title : String类型，标题
    position : String类型，消息框位置，bottom或right
    content : String类型，消息内容，可以使普通字符串或HTML
    isshake : Boolean类型，[可选]弹出窗口后是否抖动，不支持IE9
    delay : Number类型，[可选]弹出窗口后是否延迟自动关闭，单位秒
    click : Function类型，[可选]窗体点击回调，可以根据回调方法参数获取具体点击对象
    }

    Returns
    String

    id 弹出框的id
