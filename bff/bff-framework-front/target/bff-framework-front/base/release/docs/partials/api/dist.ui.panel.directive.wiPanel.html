wiPanel
directive in module dist.ui.panel

Description

wiPanel 是基本的布局组件，可以自定义标题和按钮，并内置了一些工具按钮。

Usage

as element:
<wi-panel
       [wiid="{object}"]
       [width="{number|length}"]
       [height="{number|length}"]
       [heading="{string}"]
       [headicon="{string}"]
       [headtools="{string|array}"]
       [center="{boolean}"]
       [is-open="{boolean}"]
       [collapsible="{boolean}"]
       [collapsed="{boolean}"]
       [onopened="{function}"]
       [onclosed="{function}"]
       [oncollapse="{function}"]
       [onexpand="{function}"]>
</wi-panel>

Parameters

ParamTypeDetails
wiid
(optional)
object

若定义了此属性，可供调用接口。
.options() 获取当前设置：{ heading，isOpen，isCollapse }，
.element() 获取当前 panel 对应的 jqlite 元素，
.toggle() 切换当前面板的折叠状态，并返回切换后的状态，true 为折叠。

width
(optional)
numberlength

面板宽度，默认为 100%。
number 将自动添加单位 px。
length 为 number 接长度单位（相对单位和绝对单位）。
相对单位：em, ex, ch, rem, vw, vh, vm, %
绝对单位：cm, mm, in, pt, pc, px

height
(optional)
numberlength

面板高度，默认为自适应。若定义了此属性，head 部分高度将自动设置为 30px。 说明同 width。

heading
(optional)
string

头部显示文字, heading是单向绑定的，可以设置为一个确定的字符串，或者设置为scope中的一个变量。

headicon
(optional)
string

头部左侧显示图标的 url。

headtools
(optional)
stringarray

头部右侧的工具，系统工具：'close'（关闭），collapse'（折叠）。
string：用“,”连接的系统工具字符串，定义时需加引号，如：headTools="'close,collapse'"或"'close'"。
array：数组的每一个元素代表一个工具，系统工具为 string，自定义工具为 Object。如：['collapse','close',{name:'save',cls:'icon-save',opt:saveFun}]。
其中自定义工具对象的属性说明：name-操作名称,cls-class名称,opt-click的操作（值为函数引用，参数：事件源 event）。

center
(optional)
boolean

true 时居中显示，默认为 false

isOpen
(optional)
boolean

true 时面板显示，双向绑定，应设置为scope中的一个变量。

collapsible
(optional)
boolean

true 时可以通过点击头部进行折叠展开操作，默认为 false。

collapsed
(optional)
boolean

true 时折叠，默认为 false。

onopened
(optional)
function

打开面板后的回调方法，参数：无。

onclosed
(optional)
function

关闭面板后的回调方法，参数：无。

oncollapse
(optional)
function

折叠面板时的回调方法，参数：无。

onexpand
(optional)
function

展开面板时的回调方法，参数：无。
