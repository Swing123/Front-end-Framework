wiMenu
directive in module dist.ui.menu

Description

wiMenu 是菜单，其中可以包含一个 DOM 元素以触发弹出固定菜单。

Usage

as element:
<wi-menu
       [wi-right-menu="{boolean}"]
       [position="{string}"]
       [adaptable="{boolean}"]
       [filterable="{boolean}"]
       dataprovider="{JSON}"
       [selectitem="{object}"]
       [onselect="{function}"]>
</wi-menu>

Parameters

ParamTypeDetails
wiRightMenu
(optional)
boolean

右键菜单标识。固定位置的菜单可省略此属性，或设置为"false"。

position
(optional)
string

固定位置的菜单弹出方向，右键菜单此属性无效。
默认为 "bottom-left"。可选值：
"bottom-left"/"bottom-right"/"top-left"/"top-right"/"left-top"/"left-bottom"/"right-top"/"right-bottom"

adaptable
(optional)
boolean

第一级菜单是否根据 DOM 可视区域调整弹出方向，默认为 true。

filterable
(optional)
boolean

第一级菜单是否可根据关键字筛选，默认为 false。

dataproviderJSON

菜单数据源，绑定项，只能根据当前 scope 中的变量控制。数据项：
id: string 菜单项 id
label: string 菜单项显示内容
icon: string 可选，菜单项自定义图标
filterable: boolean 可选，子菜单是否支持搜索
children: [] 可选，子菜单数据源
enabled: boolean 可选，菜单项是否可用
event: string 可选，可直接执行的表达式（复杂事件根据点击后的选中菜单项自定义）

selectitem
(optional)
object

整个菜单中选中的数据项，绑定项，只能根据当前 scope 中的变量控制 - 将删除此方法。

onselect
(optional)
function

选中菜单项后执行的自定义方法，参数为选中项。
