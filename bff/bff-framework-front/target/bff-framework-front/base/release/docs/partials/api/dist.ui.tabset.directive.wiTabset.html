wiTabset
directive in module dist.ui.tabset

Description

标签页组件标签，内部可包含wiTab标签页定义

Usage

as element:
<wi-tabset
       [width="{number|length}"]
       [height="{number|length}"]
       [vertical="{boolean}"]
       [tabheadsize="{pixels}"]
       [closeable="{boolean}"]
       [enablerightmenu="{boolean}"]
       [wid="{object}"]
       [onbeforeclose="{function}"]
       [onclose="{function}"]
       [onselect="{function}"]>
</wi-tabset>

Parameters

ParamTypeDetails
width
(optional)
numberlength

组件宽度，默认为 100%。
number 将自动添加单位 px。
length 为 number 接长度单位（相对单位和绝对单位）。
相对单位：em, ex, ch, rem, vw, vh, vm, %
绝对单位：cm, mm, in, pt, pc, px

height
(optional)
numberlength

组件高度，默认由内容撑开。
说明同 width。

vertical
(optional)
boolean

标签页是否纵向排列，默认为 false。

tabheadsize
(optional)
pixels

标签页头高度（纵向为宽度）(不含单位：px)，默认为 36。

closeable
(optional)
boolean

所有标签（非固定）支持关闭，默认为 true，此时不支持禁用标签（wi-tab 中 disable 属性失效）。

enablerightmenu
(optional)
boolean

是否启用标签页右键菜单，默认为 false。若 closeable=false，此属性失效。

wid
(optional)
object

若定义了此属性，可供调用接口。调用方法参见：wid.

onbeforeclose
(optional)
function

关闭标签页前的回调方法名，该方法若返回 false 将阻止关闭，参数为标签对应的 data：

  • wiid: 初始化 wi-tab 指令时设置的 wiid 属性值
  • src: 初始化 wi-tab 指令时设置的 src 属性值
  • icon: 初始化 wi-tab 指令时设置的 src 属性值
  • heading: 初始化 wi-tab 指令时设置的 heading 属性值，或 wi-tab-heading 中的html字符串
onclose
(optional)
function

关闭标签页后的回调方法名，参数同 onbeforeclose.

onselect
(optional)
function

标签选中后的回调方法名，返回 false 时将阻止关闭，参数同 onbeforeclose.
