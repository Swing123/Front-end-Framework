wiDatagrid
directive in module dist.ui.datagrid

Description

数据表格

Usage

as element:
<wi-datagrid
       [wiid="{object}"]
       [dataprovider="{array}"]
       [dataurl="{string}"]
       [params="{string}"]
       [pagemode="{string}"]
       [itemclick="{function}"]
       [itemdoubleclick="{function}"]
       [columns="{object}"]
       [lockcolumns="{number}"]
       [rowcolorfunction="{function}"]
       [showno="{boolean}"]
       [multiselect="{boolean}"]
       [treefield="{string}"]
       [collapse="{boolean}"]
       [itemopen="{function}"]
       [width="{number|percent}"]
       [height="{number|percent}"]
       [headerheight="{number}"]
       [rowheight="{number}"]
       [wordwrap="{boolean}"]
       [pageselect="{array}"]
       [showexcel="{boolean}"]
       [showrefresh="{boolean}"]
       [pagebar-height="{number}"]
       [pagebarrenderer="{string}"]>
</wi-datagrid>

Parameters

ParamTypeDetails
wiid
(optional)
object

供外部调用.

PropertyTypeDetails
getSelectedItems
(optional)
function

获取选中条目 多选.

getSelectedItem
(optional)
function

获取选中条目 单选.

dataprovider
(optional)
array

数据源，若定义了 dataurl，此属性失效.

dataurl
(optional)
string

请求数据的url，若定义了此属性，dataprovider 失效.

params
(optional)
string

请求数据的参数（lx, keyword, start, limit）.

pagemode
(optional)
string

分页模式：client:前台分页 server：后台分页 none:不启用分页(将隐藏分页工具条).

itemclick
(optional)
function

单击事件.（itemclick和itemdoubleclick不能同时设置）

itemdoubleclick
(optional)
function

双击事件.（itemclick和itemdoubleclick不能同时设置）

columns
(optional)
object

通过数据源实现的列定义.

lockcolumns
(optional)
number

锁定前几列.

rowcolorfunction
(optional)
function

设置行颜色.

showno
(optional)
boolean

显示行序号，默认为 false，不显示序号.

multiselect
(optional)
boolean

多选，默认为 false，不显示复选框.

treefield
(optional)
string

树节点字段.

collapse
(optional)
boolean

初始化时树型节点是否折叠，false 时展开，默认为 true。

itemopen
(optional)
function

节点展开，动态获取子节点数据 children[]。itemopen(data, callback))
参数：data 要展开的节点数据，callback 回调函数，获取 children[] 后，需手动执行 callback(data, children[]).

width
(optional)
numberpercent

宽度.指令直接从属性值attrs['width']获取数字(不含单位：px)或百分比.

height
(optional)
numberpercent

高度.指令直接从属性值attrs['height']获取数字(不含单位：px)或百分比.

headerheight
(optional)
number

表头的行高.(不含单位：px),默认为 50.

rowheight
(optional)
number

表内容的行高.(不含单位：px),默认为 40.

wordwrap
(optional)
boolean

单元格内容是否支持自动换行，默认为 false，若定义了 lockcolumn，此属性失效。

pageselect
(optional)
array

每页显示数量.默认为：[20,30,40,50,60]

showexcel
(optional)
boolean

显示excel按钮.

showrefresh
(optional)
boolean

显示刷新按钮.

pagebarHeight
(optional)
number

分页栏高度.(不含单位：px),默认为 30.

pagebarrenderer
(optional)
string

自定义工具条.
