wiPopupbutton
directive in module dist.ui.popupbutton

Description

弹出按钮

Usage

as element:
<wi-popupbutton
       [label="{string}"]
       [isopen="{boolean}"]
       [width="{number|length}"]>
</wi-popupbutton>

Parameters

ParamTypeDetails
label
(optional)
string

按钮名称.

isopen
(optional)
boolean

是否展开.

width
(optional)
numberlength

popupbutton 的宽度.
number 将自动添加单位 px。
length 为 number 接长度单位（相对单位和绝对单位）。
相对单位：em, ex, ch, rem, vw, vh, vm, %
绝对单位：cm, mm, in, pt, pc, px
