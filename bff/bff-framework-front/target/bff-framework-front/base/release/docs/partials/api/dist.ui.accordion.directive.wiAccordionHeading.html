wiAccordionHeading
directive in module dist.ui.accordion

Description

wiAccordionHeading只能作为wiAccordionGroup的子指令来使用，它可以使用一个html片段来设定面板的头部显示。如果你希望头部显示的不仅仅是简单的文字， 比如还有图片，按钮等更复杂的显示，就需要使用到这个子指令。使用方法见demo文档。

Usage

as element:
<wi-accordion-heading>
</wi-accordion-heading>
as attribute
<ANY wi-accordion-heading>
   ...
</ANY>
